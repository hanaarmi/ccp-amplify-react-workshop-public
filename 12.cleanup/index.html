12. 인프라 정리 및 Budget 설정

Event Engine 을 사용하여 실습을 진행한 경우는 별도로 리소스 정리를 하실 필요가 없습니다. 개인 Account 를 만들어서 진행한 경우에는 아래 리소스를 정리해야 더 이상 과금이 되지 않습니다. 더 실습을 진행하고자 하실 경우, 적당한 예산을 설정하여 원치 않는 금액의 과금이 되지 않도록 메일 알람이 오도록 할 수 있습니다.

Budget 설정

  1. AWS Budgets 서비스로 이동합니다.

  2. Create a budget 버튼을 클릭하여 예산을 생성합니다.

  3. Cost budet 을 선택하고 Next 버튼을 클릭하여 진행합니다.

  4. 아래와 같이 적당한 예산을 설정하여 진행합니다.

  5. 예상되는 예산의 Alert 이 오도록 Threshold 를 설정하고 알림을 받을 이메일 주소를 설정하고 진행합니다.

  6. Next 버튼을 클릭하여 진행합니다.

  7. Create budget 을 클릭하여 설정을 마무리합니다.

  8. 이제 정해진 예산의 설정된 Threshold 를 넘길 경우 메일로 알림을 받을 수 있습니다.
