Event Engine 사용하기

이벤트 형태로 워크샵을 진행하는 경우 Event Engine 을 사용해서 워크샵을 진행할 수 있습니다. (정확한 내용은 진행자에게 확인하시기 바랍니다.) Event Engine 안내를 별도로 받지 않으신 경우에는 바로 전단계인 Account 만들기 를 통해 시작하시기 바랍니다.

  1. Event Engine Dashboard 로 이동합니다. 아래 화면이 보입니다. 진행자가 hash code 를 안내해주면, 아래 화면의 빈 칸에 hash code 를 입력하고 아래 버튼을 눌러 진행합니다.

  2. 인증 단계입니다. Email One-Time Password 방식을 선택합니다.

  3. 개인 인증을 할 개인 이메일 정보를 입력합니다. (워크샵 이벤트 이전에 이메일 주소를 진행자에게 보내셨다면, 해당 이메일 정보를 사용하셔야합니다.) 아래 버튼을 눌러 계속 진행합니다.

  4. 이메일을 통해 받은 Passcode 를 입력합니다.

  5. 아래 화면에서 AWS Console 버튼을 누릅니다.

  6. 아래 화면에서 Open AWS Console 을 누르면 콘솔에 진입할 수 있습니다. 이제 AWS 인프라를 사용하실 수 있습니다. 오른쪽 Copy Login Link 를 클릭하면, 로그인 URL 주소가 복사됩니다. 이후 위와 같은 과정을 거치지 않아도 해당 URL 을 사용하여 로그인할 수 있습니다.

  7. AWS 콘솔 화면입니다. 계속해서 1번에서 안내해드린 Event Engine Dashboard 페이지에서도 다시 로그인이 가능하며, 6번에서 안내해드린 Login Link 를 통해서도 접속이 가능합니다.

이벤트 엔진은 보통 이벤트 워크샵 기간 동안만 활성화되는 Account가 발급됩니다. 워크샵이 종료되면 모든 인프라가 삭제되며 해당 Account 에 다시 접속할 수 없습니다. 정확한 유지 기간은 진행자에게 문의해주세요.
