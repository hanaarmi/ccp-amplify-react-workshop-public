1. 사전 준비

이번 챕터에서는 Connect 인스턴스를 생성해봅니다. Immersion Day 를 Event Engine 을 통해서 진행하실 경우는, Event Engine 사용하기 를 참고하시고, 그렇지 않은 경우에는 Account 만들기 를 참조하시고 Account 를 직접 만드시기 바랍니다. 구체적인 진행 방법에 대한 안내는 진행자에게 문의해주세요.

Account 만들기 와 Event Engine 사용하기 중 하나의 방법만 사용하시고 이후 Connect Instance 생성 으로 이동하면 됩니다.
