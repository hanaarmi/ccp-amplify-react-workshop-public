8. Customer Profiles 기능 추가하기

Amazon Connect Customer Profiles 는 여러 외부 애플리케이션의 고객 정보를 Amazon Connect의 연락처 기록과 빠르게 결합할 수 있도록 사전 구축된 통합을 제공합니다. 이를 통해 고객 상호 작용 중에 에이전트에게 필요한 모든 정보를 한 곳에서 확인할 수 있는 고객 프로파일을 만들 수 있습니다.

Customer Profiels 설정하기

  1. Amazon Connect 서비스로 이동하여 생성한 인스턴스를 선택합니다.

  1. 왼쪽 메뉴에서 Customer Profiles 를 선택하고, Enable Customer Profiles 버튼을 클릭합니다.

  1. Create new domain 을 선택하고 Domain name 을 입력합니다. 고객 정보를 관리할 회사 이름이나 특정 목적을 잘 알 수 있도록 이름을 설정하면 됩니다.

  1. Customer Profiles 는 모든 고객 정보를 암호화하여 저장합니다. 따라서, 암호화 Key 를 만들어야합니다. 암호화 Key는 KMS 서비스를 통해 생성/관리할 수 있습니다. 아래 버튼을 클릭하여 KMS Key 를 생성해보도록 합니다.

  1. Symmetric Key type 을 선택하고 Next 버튼을 클릭합니다.

  1. Alias 이름을 입력하고 Next 버튼을 클릭합니다.

  1. Define key administrative permissions 페이지는 Next 버튼을 클릭하여 넘어갑니다.

  2. Define key usage permissions 페이지는 Next 버튼을 클릭하여 넘어갑니다.

  3. 아래와 같이 Key 가 만들어졌습니다.

  1. 다시 Customer Profiels 를 생성하던 Tab 으로 이동합니다. Specify KMS key box 에서 방금 전 생성한 Key 를 선택합니다.

  1. Enable Customer Profiles 버튼을 클릭하여 생성을 마무리합니다.

  1. Customer Profiles domain 정보와 KMS Key 를 확인합니다.

  1. src/component/ccp/ccp.component.jsx 파일의 componentDidMount() 함수를 아래와 같이 수정합니다.
async componentDidMount() {
    const { attributes } = await Auth.currentAuthenticatedUser()
    console.log(attributes.email)
    ccpAuthLogin(attributes.email).then((result) => {
        console.log(result.ccpResult)
        this.setState({ credentials: JSON.stringify(result.ccpResult) })
        this.loginFrm.current.submit()

        window.connect.agentApp.initApp(
            'ccp',
            'ccp',
            'https://[Connect인스턴스URL]/ccp-v2/'
        )
        window.connect.agentApp.initApp(
            'customerprofiles',
            'customprofile',
            'https://[Connect인스턴스URL]/customerprofiles-v2/'
        )

        window.connect.contact(function (contact) {
            contact.onConnected(function () {
                document.getElementById(
                    'logMsgs'
                ).innerHTML = `onConnected(${contact.getContactId()})<br>`
                var attributeMap = contact.getAttributes()
                var note = JSON.stringify(attributeMap['note']['value'])
                var phonenumber = JSON.stringify(
                    attributeMap['phonenumber']['value']
                )
                var firstname = JSON.stringify(
                    attributeMap['firstname']['value']
                )
                var lastname = JSON.stringify(
                    attributeMap['lastname']['value']
                )
                console.log(attributeMap)
                console.log(note)
                document.getElementById(
                    'logMsgs'
                ).innerHTML += `${note}<br>${phonenumber}<br>${firstname}<br>${lastname}<br>`
            })
        })
    })
}
